Projects

Drive Map

C# and Windows Forms

This was a quick project that was assigned to me by my supervisor. This application was used by one of my professors. The user puts their username and password for their directory and the application maps the user's drive and gives them full permission. The user does not have to navigate the file explorer to go to their directory.

Folder Security Permission (Console)

C#

This project was the first part of a long-term project. This application is going to be utilized by the I.T. Department. The project asks the user which drive they want to create their folder in, asks the name of the directory, and sets permissions for the current user. If the directory name exists, it asks the user if they want to modify the folder's perissions or delete the directory. I added a lot of error and exception handlers to make sure the application does not have any bugs or produce errors.

Folder Security Permission (GUI)

C# and Windows Forms

This project was the second part of the long-term project. The project is basically the same as the console version, but using a graphical user interface. I also added a feature that lets the current user set a permission for a specific person. Additional exception and error handlers are added.

Folder Security Permission (GUI Version 2)

C# and Windows Forms

This project was the last part of the long-term project. This project expanded with more features. It selects the drive, lists the contents of it and lists the user's permissions on a specific folder. More exception and error handlers are added. I am planning on migrating this project to UWP or WPF.
